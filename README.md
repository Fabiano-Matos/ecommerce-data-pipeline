# Pipeline da Dados de Comércio Eletronico 
Este projeto tem como objetivo praticar conceitos de Engenharia de Dados
por meio da construção de um pipeline ETL utilizando dados de ecommerce 

# Objetivo 
O pipeline foi estruturado para seguir as etapas de:
**Extração** dos dados a partir de um arquivo CSV
**Transformação** para limpeza, padronização e preparação dos dados 
**Carga** dos dados tratados para uso posterior em análises ou armazenamento 

# Estrutura do projeto 
pipeline
│
├── dados
│   ├── brutos        # dados originais do dataset
│   └── processados   # dados tratados pelo pipeline
│
├── src
│   ├── extrair       # scripts de extração dos dados
│   ├── transformar   # scripts de limpeza e transformação (em desenvolvimento)
│   └── carregar      # scripts de carregamento (em desenvolvimento)
│
└── README.md
